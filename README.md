# DevOps Task Manager

A modern task management application built with React and Vite, designed to demonstrate DevOps best practices including CI/CD, automated testing, and deployment workflows.

## Tech Stack

- **Frontend**: React 18 + Vite
- **Testing**: Jest, React Testing Library, Playwright
- **CI/CD**: GitHub Actions
- **Deployment**: AWS EC2

## Getting Started

```bash
npm install
npm run dev
```
